# Pixlutter for Ruby

The jolly file viewer for pixtures by Ruby/Clutter.

## Installation

Add this line to your application's Gemfile:

    gem 'pixlutter'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install pixlutter

## Usage

    % pixlutter [FILE]...
